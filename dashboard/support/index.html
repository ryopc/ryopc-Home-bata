よくある質問（FAQ）
ryopcとは何ですか？
ryopc（全国PCサポートネットワーク）は、PC・スマホ・インターネット・AI活用などをサポートする技術プロジェクトです。 Web制作、AI導入、サーバー構築、トラブル対応など幅広く対応しています。
どんなことを相談できますか？
以下のような内容に対応しています：

・パソコンの不具合
・Wi-Fiやネット接続
・スマホ設定
・Webサイト制作
・AI導入・活用
・Cloudflare / GitHub / WordPress 構築
・サーバー・ドメイン設定
・Scratchやプログラミング学習

※内容によっては対応できない場合があります。
サポート料金はいくらですか？
基本無料です。
一部の高度な制作・構築作業などは別途相談となる場合があります。
オンライン対応はできますか？
可能です。
チャット・メール・画面共有などを利用してオンラインで対応しています。

すべてオンライン対応となります。

安全な利用を前提としてサポートしています。
